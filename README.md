# HP分析ツール 

URLを入力すると、HPが「問い合わせにつながる状態か」を診断する簡易Webアプリです。

## 主な機能

- HP URL診断
- 競合URL比較
- PDFレポート出力
- お客様提示用の詳細診断
- 重要減点項目の表示
- 画像バナー依存・古いHTML構造の簡易検出
- スマホで使いやすいUI（入力フォームをメイン画面化、結果をタブ表示）

## 使い方

```bash
cd /Users/weby/Downloads/hp_analyzer_mvp
python3 -m pip install -r requirements.txt
python3 -m streamlit run app.py
```

ブラウザで以下を開きます。

```text
http://localhost:8501
```

## スマホでの使い方

- 入力フォームはサイドバーではなく、メイン画面上部に表示されます。
- 競合URL入力や詳細設定は「詳細設定・競合比較（任意）」を開いて入力します。
- 診断結果は「サマリー / 改善ポイント / 競合比較 / 基本情報」のタブで確認できます。
- iPhoneやAndroidではブラウザの「ホーム画面に追加」を使うと、アプリのように起動できます。

## 診断レベル

画面内の「診断レベル」で変更できます。

- 詳細診断：重要項目まで確認する厳しめの診断。初期設定。
- 通常診断：標準的な確認項目で診断。
- 簡易診断：単純な構成チェックに近い診断。

お客様に見せる場合は「詳細診断」のままで問題ありません。

## 注意点

このツールはURLから取得できる情報に基づく簡易診断です。実際のアクセス数、問い合わせ数、CVR、広告成果、検索順位はGA4、Search Console、Google広告、GBP等との連携が必要です。
